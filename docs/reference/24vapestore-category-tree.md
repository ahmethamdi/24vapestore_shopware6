## Mimari Notlar / Açık Sorular

1. **Marka = kategori mi, property mi?** Mevcut site markaları kategori ağacına gömmüş.
   Bu, 4000 SKU'da filtreleme UX'ini zayıflatır ve ağacı devasa yapar. Alternatif:
   markalar property group / manufacturer, kategoriler ürün tipine göre
   (Pods, Akkuträger, Liquids, Einweg...). Karar verilmeli.
2. **Tekrar eden markalar:** Elf Bar, Lost Mary, Al Fakher, Vozol, SALT birden fazla ana
   kategoride görünüyor. Kategori ağacında tekrar = bakım yükü; property olsa tek yerde durur.
3. **3. seviye tip ayrımı** (Pods / Akkuträger / Sets) aslında bir **ürün tipi property'si** —
   her markanın altında tekrar ediyor.
4. **Jugendschutz** (yaş doğrulama) footer'da var — akış incelenmeli, bizde de gerekli.
5. **Händler-Login** B2B ihtiyacına işaret ediyor; brief B2C dedi, netleştirilmeli.
